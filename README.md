# タスクリスト PWA

## 使い方

### ローカルで起動する（必須）
PWAはHTTPSまたはlocalhostで動作する必要があります。
`index.html` をダブルクリックしても動作しません。

#### Node.jsがある場合
```bash
npx serve .
```
ブラウザで http://localhost:3000 を開く

#### Pythonがある場合
```bash
python3 -m http.server 8080
```
ブラウザで http://localhost:8080 を開く

### デスクトップにインストール
1. 上記の方法でブラウザで開く
2. アドレスバー右側の「インストール」アイコンをクリック
   （またはページ内の「デスクトップにインストール」ボタン）
3. インストール後はデスクトップのアイコンから直接起動できます

## アイコンについて
`icons/` フォルダに以下のPNG画像を用意してください：
- `icon-192.png`（192×192px）
- `icon-512.png`（512×512px）

なくてもアプリは動作しますが、インストール時のアイコンがデフォルトになります。
